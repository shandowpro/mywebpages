    3. { [productId] } Dynamic Page [index.js] :

      This page/file is created to navigate to more details product page [by clicking on {show more Details} button in the each product list ], in order to display more details about the current product {NewProductForm.js} file :

      This page router is created as dynamic Router, [so it created by using {product ID} ]

      Due this Router is created to be activated by clicking on the button , not as [Link] => SO we need to define this router manually , not as default router as we do in the link [inside the impoted functional component {ProductItem} ]

      Due we use a static data of products , we will assgning product data [props] values manually , just for coneptual dispalying , but will re-assgning these [props] values by using dynamic data from backend database in future in next lesson [using the {End Points} & {Pre-Rendering} ]

      A] Define the page Router functional compment of the :

      // Importing/calling component function {} :

      import ProductDetails from '../../components/products/ProductDetails'; => [this component has been improted to use it's passes props to be diplayed in this page ]

      function ProductDetailsPage() {
      return (
      <>
      // calling the component [ProductDetails] and assigning thier [props] values manually :
      < ProductDetails => [note that the [Id] product is not considered to be dispalyed , and not of passed props ]
      image : 'url' ,
      name : 'apple' ,
      price : 30 ,
      desc : 'This is an Apple'
      />
      </>
      );

    • differences between the { Normal router Navigation -usign Link || > a < -} & {button function calling Router navigation} :
      1. Concept :
        • {Normal Router link Navigation} =>

          To create a hyperlink navigator through using the hyper link {Link} || {a} element or button {using calling defined Router function } , as following :

          1. {a} =>
            • the old HTML element to create a link in web document
            • using the {href} attribute to assing the link url
            • NOT Need to import any function from the nextJS to use this element

          2. {Link} =>
            • the new JSX|TSX element to create a link in webpages => using a defined router in [NEXTJS] | [ReactJS]
            • using the {href} attribute to assing the link url
            • Need to define the builtin hock function [Link] from the 'next/link'



        • {Function Router button Navigation} =>

          To create a hyperlink navigator through {using calling a function of Defined Router as new item pushed into the list of Routers in the NextJS } , as following :

          1. Need to import the {useRouter} builtin form nextJS built-in functions
          2. Need to define a Router variable extracted from imported {useRouter} built-in function
          3. The new Router must be pushed as new item into the define Router variable arary list , all of this inside the button ruoter definition function
          4. to implement this function =< must be called inside a button within the using event [onClick{}] as JSX element


      2. script & Syntax :
        • {Normal Router link Navigation} => by using (2) methods/elements :

          1] using [new method] of ReactJs {Link} JSX element , as following :
          Importing the built-in [Link] from the nextJs Hocks functions :
          import Link from 'next/link' ;

          Define the functional conponent :
          function funcName() {
          return(
          < div >
          < Link href={/pageRouter} > Go to link < /Link >
          < / div >
          );
          }


          2] using [old method] of ReactJs {a} HTML element , as following :
          Define the functional conponent : [no need to prevoius importing the Link hock function ]
          function funcName() {
          return(
          < div >
          < a href={/pageRouter} / > Go to link < /a >
          < / div >
          );
          }




        • {Function Router button Navigation} => by using (1) method :

          To create a hyperlink navigator through {using calling defined Router function } , as following :

          1. define the router as pushed item into the list of builtin nextJS routers
          2. to implement this function =< must be called inside a button within the using event [onClick{}] as JSX element


      3. Usages and impoortance :
        • {Normal Router link Navigation} =>

          used commonly to create a {huyperlink element} in the web page

          used commonly to create a huyperlink within {Navbar element} as an items list

          used commonly to create a huyperlink of {Normal Router}



        • {Function Router button Navigation} =>

          used commonly to create a huyperlink {Button element} in the web page

          used commonly to create a huyperlink within {item inner page}

          used commonly to create a huyperlink of {Dynamic Router} to navigate to a certain current item
