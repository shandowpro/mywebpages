4 Routers and Links in NextJS concept :

Concept using [Dynamic Router] within [Links] =>

  • concept of the usage of the [dynamic router] using a [Link] reactJs element => is to dynamic navigate to the clicked item according to the ID value of the clicked Item
  • The used Link can hold any value to be auto assinged to navigate path



Practical steps of using [Dynamic Router] within [Links] =>

  • Creating the [Dynamic Router File] inside the parent inner page
  • Creating the script of the dynamic page , includting the definition of router variable
  • Creating the [Links] inside the another inner page(s) to navigate to the selected item according [dynamic router]



Practical syntax of using [Dynamic Router] within [Links] =>

  1. [Parent Inner page] File => {products.js}
    // 1] General importing :
    import Link from 'next/link' ;
    import {Fragment} from 'react' ;

    // 2] The [parent functional component] :
    function pageDetails() {
    return (
    // Adding fragment to the function return , to add mulitple elements :
    < Fragment >

    // Adding the page title :
    < h1 > Welcome to our products Page < /h1 >

    // Adding link to access the products by using the dynamic links [form the parent inner page ] :
    < nav >
    < ul >
    // First product Dynamic link :
    < li >
    < Link href= {"/product/product1"}/ Link >
    < /li >

    // Second product Dynamic link :
    < li >
    < Link href= {"/product/product2"}/ Link >
    < /li >

    // Third product Dynamic link :
    < li >
    < Link href= {"/product/product3"}/ Link >
    < /li >
    < ul >
    < nav >
    < / > -> [end of the fragemnt]
    ) ; }

    export default pageDetails ;



  2. [Dynamic Router page] { [productId.js] } ->
    // 1] General importing :
    import {Fragment} from 'react' ;

    import {useRouter} from 'next/router' ;

    // 2] The [child inner pqage of the dynamic router] :
    function productDetails() {
    // 3] The [dynamic router] definition :
    // Define the router variable from the imported router list built-in function :
    const router = useRouter();
    // Define the page Dynamic router variable from the imported built-in function [using the direct method ] :
    const productId = router.query.productId ;
    return (
    // using the defined dynamic router in the script :
    < h1 > Welcome to our dynamic products {productId} < /h1 >

    ) ;
    }

    export default productDetails ;
