2 setup project files and folders concept :

Main steps of project setup and Installing => How to install [next.js] on my pc and use it to create my next app folders , by using following steps :

  1. Create and access the project path [to establish my proejct inside]
  2. Check for my current version of each [Node] including ( npm , npx) , using the following code :
    node -v
    npm -v
    npx -v

  3. Creating the [nextJS] app using {Boiling Plates} steps by using [NPX] , using the following code:
    npx create-next-app

  4. The {Boiling Plates} steps configurations should including the following properties :
    • Non [App] Rouer , => we wil use [page file name || page folder name ] method
    • Non [Typescript]
    • Non [.src]
    • Non [Tailwind]


Checking project files downloaded and Installed versions :
  1. {node_modules} folder => which include all installed node packages can be implemented inm your app
  2. {package.json} folder => which include all project confiturations and verson of all libraires an frameworks , as dependencies object
  3. {pages} folder => with following properties : [will be studied in detailed in the next lesson ]
    1. This folder is instead of using [app folder methodology] in some installation cases of [Boiling plate] setup steps , which will NOT be used in this app
    2. This folder is the most important one in the nextJs Appciation , due it includes all project pages [main(home) and inner ]
    3. This folder will be used to create the page Routers , whether using the :
      1. [file Router] method , and will include in this case => {index.js & pageName.js} files ]
      2. [folder Router] method , and will include in this case => {pageName => {index.js} file } ]
  4. {public} folder =>
    1. Include the icons files of the project
    2. Was Not existed in using ReactJs [without NextJS] , and would created automaticallly by contacting with server , by SPA
    3. Was including the [index.js] , but with NextJS [index.js] file exitsted in the main page folder , due using the [Pre-renedering] in the nextjs
  5. {styles} folder =>
    1. Include main css files public: [globals.css(still existed in the [app Router] method )]
    2. Include the Home page: [Home.Modules.css (instead of page.module.css) in the [app router]]
    3. Include the Inner page:[pageName.css]
  6. {.gitignore.} file =>
    1. was existed in the ReactJS without using the nextJs
    2. Include the ignored files and folder in your application
    3. This file is helpful to prevent certain files and folders to be uploaded to the server such as [github]
    4. The most common files to be ignored :
      • [node_modules] folder
      • [.env*.local] file -> of private flles list
  7. {next.config.js} file =>
    1. Includes all project Rules within NextJs
    2. Will be studied later
  8. {README.md} file =>
    1. Readme file for github repository uploading
  9. {.eslintrc.json} file =>
    1. will be taken later

npm i
