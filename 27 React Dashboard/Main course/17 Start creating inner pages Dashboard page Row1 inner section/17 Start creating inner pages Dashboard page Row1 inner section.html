17 Start creating inner pages Dashboard page Row1 inner section concept :

A] The main segmentation of [Dashboard.jsx] inner page :

The main segmentation of the [Dashboard.jsx] as (3) rows the following :

  • [Download Report] button
  • {Row1} => The first Row after the downlaod report
  • {Row2} => The second Row after the {Row1}
  • {Row3} => The third Row after the {Row2}



B] [Row1.jsx] the first inner section as the first row of [Dashboard.jsx] :

The main segmentation of the [Row1.jsx] as (4) imported Reusable inside a container parent [Stack] the following :

  • 1] Importings :
  • The [Mui] imported elements -> to be used inside the main [Row1] component
  • The defined {useTheme} hook -> to be used to call the builtin palette colors
  • The used Mui icons -> to be used inside each card Component as an iconic element value of the 'icon' attribute
  • The Defined all datasources variables from the [data] file , as the datascources values of 'data' attribute

  • 2] Hooks :
  • {useTheme()} hook to be used in calling the Builtin [palette] for coloring
  • 3] Main contents inside the return :
  • {Stack} parent container - for automatic display 'flex' value - of all it's contents , with following attriubtes :
    1. direction={"row"} => to align all elements inside it as 'row' direction instead of the column dirction as the default
    2. flexWrap={"wrap"} => to wrap the elements in case of smaller screen
    3. gap={ 2 } => to take a space between the elements
    4. justifyContent={ { xs: "center", sm: "space-between" } } => to set the elements in (2) defferent mode according to the current screen scale

  • The children elements inside the {Stack} parent container => (4) Reusable imported [Card] from external file



C] The Re-Usable component {Card.jsx} the seperated and exporterd into the first inner section [Row1.jsx] of the [Dashboard.jsx] :

The reusable component to be created inside an external and being imported inside the main inner section component {Row1} , with following properties :

  • Sent children properites with the functional component , following :
    1. {icon} => [ will be filled by defined Iconic MUI variable within the main inner section component { Row1.jsx } ]
    2. {title} => [ will be filled a string text ]
    3. {subTitle} => [ will be filled a string text ]
    4. {data} => [ will be filled a defined and imported variables from external file ]
    5. {increase} => [ will be filled a string text ]
    6. {color} => [ will be filled by string value of the 'Scheme' attriubte of the {ResponsivePie} chart element - which refers to the chart color theme - ]
