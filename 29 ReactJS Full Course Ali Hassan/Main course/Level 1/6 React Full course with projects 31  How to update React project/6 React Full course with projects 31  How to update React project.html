ReactJS [6 SideBar Segmentation] Main Topics :

> Main Subjects in this lesson will be => main Create [SideBar] component's inner components of [Admin dashboard] project , as following :

Main Subjects :

  1. Main points of this lesson , Create SideBar's Inner components (3) in {SideBar.jsx} , as following :
    1. A] First section [in SideBar.jsx] :
      1. 1- { Avatar } and image of user section
      2. 2- { Typography } for the user name
      3. 3- { Typography } for the user title

    2. B] Second & Third & Fourth sections [in SideBar.jsx] :
      1. Defined array of the objects for each section [3 arrays] to be used in mapping through (3) items for each array as following :
        • Text => the title of the icon
        • icon => the imported element icon from the [MUI]
        • path => the path of the each icon inner router page -> [will be created later ]

    3. C] Set conditional Display of the SideBar's contents by using [React Routering ] :
      1. Procedures In {SideBar.jsx}
      2. Procedures In {App.jsx}
      3. Procedures In {project folders }
      4. Procedures In {Main.jsx}

    4. D] Set a conditional background color of the menu item for both them modes -> in [ SideBar.jsx ]



6 SideBar Segmentation concept :

A] First section => the user's main information section, and it include (3) items , as following steps :


  1. 1- { Avatar } for user section -> an imported component , and include the folowing :
    • User image => which will have custom properties [including : consittional value of both width and height for the image according to SideBar case (open/close) ]
    • User image source => 'src'
  2. 2- { Typography } for the user name => css properties + text
  3. 3- { Typography } for the user title => css properties + text


B] Second & Third & Fourth sections => as following steps :


Each section will map through a custom defiend array of objects , which will provide each item inside the list by each item value , which are (3) [text + icon + path ] for each item inside the section [list] :

  1. 1- First section [list] , and include the items of the first array as following :
    • Dahsboard
    • Manage Team
    • Contacts Information
    • Invoices Balances

  2. 2- Second section [list] , and include the items of the second array as following :
    • Profile Form
    • Calendar
    • FAQ Page

  3. 3- Third section [list] , and include the items of the second array as following :
    • Bar Chart
    • Pie Chart
    • Line Chart
    • Geography Chart


C] Set conditional [route] Display of the SideBar's contents by using [React Routering ] , as following steps :


  1. 1] Procedures In {SideBar.jsx} :

    Assign the path of the each page as a value of each 'path' property insde each array => by folder name of each inner page

    Importing {useNavigate} from "react-router-dom" library

    Inside the {onClick} event of [ListItemButton] element => Using the imported Hook to set a router path Hook function [of paramter value of dynamic value with in maping for each memu item ]


  2. 2] Procedures In [project files] :

    Create new folder of 'page' inside the main 'src' , wihch will include following new folders [one new folder for each inner page router] :

    • [dashboard] => include 'Dashboard.jsx' functional comppnent
    • [team] => include 'Team.jsx' functional comppnent
    • [contact] => include 'Contacts.jsx' functional comppnent
    • [invoices] => include 'Invoices.jsx' functional comppnent
    • [form] => include 'Form.jsx' functional comppnent
    • [calendar] => include 'Calendar.jsx' functional comppnent
    • [faq] => include 'Faq.jsx' functional comppnent
    • [bar] => include 'Bar.jsx' functional comppnent
    • [pie] => include 'Pie.jsx' functional comppnent
    • [line] => include 'Line.jsx' functional comppnent
    • [geography] => include 'Geography.jsx' functional comppnent

  3. 3] Procedures In {App.jsx} :
    1. Import the { < Outlet / > } form 'react-router-dom'
    2. inside tne [Box] main element -> Assign { < Outlet / > } as an imported element from 'react-router-dom' library => to be an outlet and display the clicked element from the [SideBar] element within the {'Box'} inside the assembling component
  4. 4] Procedures In {Main.jsx} :
    1. Import each page's component to be used as router inside the Main.jsx { < Main / > } form each page path
    2. Create a single router for each page inside the [router] deifnied public variable by following the next instructions :
      • The general syntax of each Router is as next code :
        < Route path={path} element={ < Component / > } />

        The {Route} main element has (2) major attributes :

        • [ path = {'path'} ] => contain the folder of the inner page as a string value 'small case'
        • [ element = {< Component / > } ] => contain the defined and imoptred compoent of the inner page as an called element


      • The main contents of the defined routes inside/wrapped the main 'Route' , as following :
        1. The [default Home] page route of the main page of {App} -> with next attriubutes :

          path -> = '/'

          element -> = { < App / > }


        2. The [custom Home] page route of a [Dashboard] that path into a the home => {dashboard/Dashboard.jsx} page -> with next attriubutes :

          index => an alternative path in to the home page , means to path into the same home page

          element -> = { < Dashboard / > }


        3. The inner route of a [Team] => {team/Team.jsx} page -> with next attriubutes :

          path -> = 'team'

          element -> = { < Team / > }


        4. The inner route of a [Contact] => {contact/Contact.jsx} page -> with next attributes :

          path -> = 'contact'

          element -> = { < Contact / > }

        5. The inner route of a [Invoices] => {invoices/Invoices.jsx} page -> with next attributes :

          path -> = 'invoices'

          element -> = { < Invoices / > }


        6. The inner route of a [Form] => {form/Form.jsx} page -> with next attributs :

          path -> = 'form'

          element -> = { < Form / > }


        7. The inner route of a [Calendar] => {calendar/Calendar.jsx} page -> with next attributes :

          path -> = 'calendar'

          element -> = { < Calendar / > }


        8. The inner route of a [Faq] => {calendar/Calendar.jsx} page -> with next attributes :

          path -> = 'faq'

          element -> = { < Faq / > }


        9. The inner route of a [BarChart] => {bar/BarChart.jsx} page -> with next attributes :

          path -> = 'bar'

          element -> = { < BarChart / > }


        10. The inner route of a [Pie] => {pie/Pie.jsx} page -> with next attributes :

          path -> = 'pie'

          element -> = { < Pie / > }


        11. The inner route of a [Line] => {line/Line.jsx} page -> with next attributes :

          path -> = 'line'

          element -> = { < Line / > }


        12. The inner route of a [Geography] => {geography/Geography.jsx} page -> with next attributes :

          path -> = 'geography'

          element -> = { < Geography / > }


    3. Add the main defined variable of {router} - that include all previous defined routes - as the value of [RouterProvider] element that wrapped inside the main function of the {React.StrictMode}

D] Set a conditional background color of the menu item for both Theme Modes (dark/light) -> in [ SideBar.jsx ] , as following steps :


Inside {SideBar.jsx} => This procedures is about get and use the current path value , to detect which of inner page is clicked , and then apply the conditional formating no this Menu item , and it requires the following steps :

  1. Import a react special hook { useLocation() } from 'react-router-dom' to be used for getting the current location 'path'
  2. Import a react color class [optional] from { "@mui/material/colors" } to be used for setting a certain color by using this imported class - with ability to assign a certain degree ( by write assigned color degree inside : grey[100] ) -
  3. Define a variable 'location' as an extracted value of the imported hook { useLocation() }
  4. Use the Defined variable { 'location' } within each mapped array of inner pages , inside the 'sx' of [ListItemButton] element as a nested conditional of both [location is equal the current path & conditional theme mode value ] : following code
    < ListItemButton
    sx={ {
    bgcolor:
    location.pathname === item.path
    ? theme.palette.mode === "dark"
    ? grey[800]
    : grey[300] : null,
    } }

  1. Install the main {MUI} library files into the project files , by using the next codes :

    { npm install @mui/material @emotion/react @emotion/styled }

    // 2- Adding the [MUI Icon library] => inside the main root directory , via next code : :
    { npm install @mui/icons-material }



[*] Adding the { React Router Dom } library into the project files :


  1. Install the main {React-Router-Dom} library files into the project files , by using the next codes :
    { npm install react-router-dom }
