B] Creating the main project using [{vite} Boiling plate] , as following steps :


  1. Creating the main folder of the project with small case name ['portfolio']
  2. Open the main project directory ['portfolio'] , by using VS code => and using the [CMD - command prompt] of the inner Vscode
  3. Using the following code [ from the official website of the vite => {vitejs.dev/guide} ] :
    // Create the main project folder structure via [React-vite] framework => inside the main directory :
    { npm create vite@latest }
    // fulfill the following properties :
    -> project name : ./ -> [ to create the project directly inside current directory ]
    -> Select a Framework from : [ Vanilla , Vue , (React) , Preact , Lit , Svelte , other ]

    -> Select a variant : [ Typescript , Typescript + SWC , (JavaScript) , JavaScript + SWC ]
    // After downloading is finsihed , route inside the [inside the main root ['portfolio'] of the project - via VS code ] & use the code to install the [node_modules ] with this project :
    // open the main project folder {portfolio} :
    { cd portfolio }

    // install the Node modules inside this path :
    { npm i } \\ {npm install }

  4. Starting the project by using following command :
    Running the project via using the [localhost] via [react-vite]
    { npm run dev }
