12 Hocks and useRef in ReactJS concept :

{ useRef() } concept in ReactJS :

  1. A Hocks built-in variable fuction within the {ReactJs} , to access to a certain HTML element
  2. useRef() can be controled by the Handeler function , to assign prodcessed to persue with the useRef variable
  3. the defined useRef() variable will be called inside the TSX HTML element to be called inside this element
  4. useRef() require to import [useRef] function from reactJS
  5. useRef() require to use certain dataype when define the useRef variable
  6. useRef() can be called by using (2) methods [when dealing with 'input' fields ]


{ useRef } methods in ReactJS to access to [input elemments] :

  • There are (2) methods to contorl the inpout element, usgin the useRef varaibles , as following :
    • First Method [using {onClick} event on a button] , as following Requirments procedures :

      [useRef] imporing built-in function code => as well as in the [useState] and [useEffeect]

      [useRef] varaible => which is basic to access to the calling element , an using the datatype of the <HTMLInputElement> & value of (null)

      [useRef] Handeler function => to be used in the button with assinged actions to persue when the button is being clicked

      [useRef] using and calling the {useRef} variable -> inside the [input] TSX DOM element it self to be access to it

      [useRef] using and calling the {} Handler function -> inside the [Button] DOM element it activate the funciton

    • Second Method [using {onChange} event on a the same element] , as followintg :

      [useRef] importing built-in function code => as well as in the [useState] and [useEffeect]

      [useRef] variable => which is basic to access to the calling element , an using the datatype of the <HTMLInputElement> & value of (null)

      [useRef] Handeler function => to be used in the same input element to persue the live changing according the value of the this Input field

      [useRef] using and calling the {useRef} variable -> inside the [input] TSX DOM element it self to be access to it

      [useRef] using and calling the {} Handler function -> inside the also in [input] DOM element it activate the funciton

{ useRef } differences between the both methods to access to the input] in ReactJS :

  • {First Method} => Access the input element by using the button and onClick with using the useRef and it;s handeler function => will persue assinged actions by clicking on the button [aned usuaaly used to submit the Form ]
  • {Second Method} => Access the input element by using only the useRef and it's handeler function , wiht using [onChange] Event => will persue assinged actions by live responding



{ useRef() } importance and usages in ReactJS :

  • Provide access to specific HTML Dom Elements in reactJs to persue a certain access/change on HTML DOM element or one it's attrbutes or properties [specially access to {input}] , to persue certain action
  • First method => used to provie submit function with in the input fields in the form [with (onClick) event]
  • Second method => used to provie submit function with in the input fields in the form [with (onChange) event]

{ useRef() } triggering types in ReactJS :

  • useRef() can be access the HTML dom element by (2) methods as following :
    1. [1] First Method => using a button element and onClick event to call the handeler fuciton to persue access :
      // a] Defining useRef main variable:
      const useRefVar = useRef< HTMLInputElement> (null)

      // b] Defining useRef variable's Handler Function :
      const inputClickHandler = () => {
      // Check if the useRef variable is defined :
      if (useRefVar.current) {
      // a] Get access to input element's Focus property :
      useRefInput.current.focus() ;

      // b] Get & print the input element's value property :
      console.log(useRefInput.current?.value) ;
      }
      }

      // c] Calling the [useRef] and it's handeler function in the Dom Element :
      function Home() {
      < main >
      // Adding an Input TSX Dom element in component [with the useRef variable] to be accessed :
      < input ref={useRefVar} />
      // Adding a Button TSX Dom element in component [with the useRef handerler funciton ] to be accessed and persuing the function by :
      < button onClick={inputClickHandler} /> Click on the button to get focus on the input </ button >
      < / main >
      }


    2. [2] Second Method => using the input element [onChange] event it's self to to persue access and assign the event function in this element :
      // a] Defining useRef main variable:
      const useRefVar = useRef< HTMLInputElement> (null)

      // b] Defining useRef variable's Handeler Function of get the input's value {onChange} :
      const inputChangeHandler = (e) => {
      // Get access and print the value of useRef variable of input :
      console.log(e.target.value) ;
      }

      // c] Calling the [useRef] and it's handeler function in the Dom Element :
      function Home() {
      < main >
      // Adding an Input TSX Dom element in component [with the useRef variable] to be accessed :
      < input ref={useRefVar} onChange={inputChangeHandler} />
      < / main >
      }
