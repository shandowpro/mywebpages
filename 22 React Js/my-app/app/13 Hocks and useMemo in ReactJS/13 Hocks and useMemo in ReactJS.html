{ useMemo() } concept in ReactJS :

  1. A Hocks built-in variable fuction within the {ReactJs} , to Stroe or save a defnied fucntion to be cal by certain condition
  2. useMemo() is increase performnanced of page loading by reducing the used memory , by not allowing to the saved function to be rederend unless the condition is achieved
  3. useMemo() require to import [useMemo] function from reactJS
  4. useMemo() require to import the usweMemo functgion from the react package
  5. useMemo() can use paramerter of a defined vairable of [useState] , to be as a condition of this useMemo variable
  6. useMemo() old deprecated in the old ReactJs , and has being updated in the nextJS

{ useMemo } importance and usages in ReactJS :

  • Used mainly to save memory performance by reducing the rendered defined funmctions by assinging a conditon to be used to persue this stored function
  • This call back function can e assignwed to a varaible to save its reutrn value , to be get and usued if needed
  • Differences between the {useMemo} call back function and {useEfect} callback in ReactJS :
    1. { useMemo } => used when there is a variable need to be stored and to be dislayed later , with no need to be rendered with any triger in the page
    2. { useEffect } => used when there is an event need to be displayed , by certain condition/dependency
