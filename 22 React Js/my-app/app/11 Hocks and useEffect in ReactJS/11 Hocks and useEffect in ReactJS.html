11 Hocks and useEffect in ReactJS concept :

{ useEffect() } concept in ReactJS :

  1. A Hocks built-in Callback fuction within the {ReactJs} , to specify the page Rendering to specific element or spefic event according to certain condition
  2. useEffect() can be used with one or multiple useState() variables as conditional changing in its value , to fire certain event
  3. useEffect() can be combined with useState() one or more variables as conditional value changing
  4. useEffect() can be using single or multiple conditions , in one function
  5. useEffect() can be using one or multiple useState varilbes as several conditions , in one function


{ useEffect() } importance and usages in ReactJS :

  • It can be used to fire a certian event or result without need to render the whole page, by making the reactJS render certain partial part of the component
  • It can be used to make the reactJS render certain partial part of tge component
  • It can be related with another useState variables to act certain event according the change of the these variable values


{ useEffect() } syntax :
useEffect( () => {code event(s) to be fired or persued }, [condition(s) dependencies list] )



{ useEffect() } triggering types in ReactJS :

  • useEffect() can be Trigger the assigned event(s) by several methods as following :
    1. 1] Fire the assigned useEffect event(s) by {each time the page will render} , as following example code :
      // Defining useEffect() callback function :
      useEffect(() => {
      event code to be fired
      } )


    2. 2] Fire the assigned useEffect event by {only first time the page will render }, as following example code :
      // Defining useEfefct() callback fucntion :
      useEffect(() => {
      event code to be fired
      } , [] )


    3. 3] Fire the assigned useEffect event by {acheiving single or several conditions} , as following exmaple coded :
      // Defining useEfefct() callback function to be triggerd by {single variable change condtion} :
      useEffect(() => {
      event code to be fired
      } , [definedVar] )

      // Defining useEfefct() callback function to be triggerd by {several variables change condtion [var1 or var2]} :
      useEffect(() => {
      event code to be fired
      } , [definedVar1 , definedVa2] )




    • VIP Notes to use { useEffect() } in ReactJS :

    • useEffect() with codition can be as :
      1. Change in defined single or multipe variable(s) value(s)
      2. Logic condition single or multipe variable(s) value(s)
      3. Limit value of defined single or multipe variable(s) value(s)
    • The useEffect() with codition will be triggered for thre first time even if the condition is not acheived yet
