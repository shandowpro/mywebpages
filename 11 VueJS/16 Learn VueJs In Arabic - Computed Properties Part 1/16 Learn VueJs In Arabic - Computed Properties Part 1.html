[Computed Properties] Concept:

=> [Computed Properties] Concept :
  • It is a kind of Vue Instances , and it is methods can used as Cashed Variable to provide Best Performance by avoiding Repeating all Metods and Storing the Return Value in a serperated Cash Value
  • Defined Methods in {Comptued Instance} can be Called in the Vue HTML Elements as [Variables]
  • Defined Methods has NO Parameters (as it in normal functions) , because it is Cashed Value
  • Defined Methods Not responding to the User Events and Actions, but it is responding to any Independencies of it's Logic
  • Main Differences between {Computed Methods} & {Normal Methods}
    1. Called in Vue HTML Element Syntax :
      • {Computed Methods} => As Single Variable => { { funName } }
      • {Normal Methods} => As Normal function => { { funName() } }
    2. Dealing with Parameters (Arguments) Syntax :
      • {Computed Methods} => Not has Parameters in it's Building Sytnax
      • {Normal Methods} => Could has Parameters in it's Building Sytnax as usual
    3. Called by Events & Reponding to User Action :
      • {Computed Methods} => Not Built to be called by Events or to respond to user Action
      • {Normal Methods} => Built to be called by the Events and tesponding to both user Actions
    4. Responding to Change in Function Logic's Independencies:
      • {Computed Methods} => Reponding and Re-calculate Directly to any change in logic Independencies, with only one Rendering Process
      • {Normal Methods} => Reponding and Re-calculate Directly to any change in logic Independencies, BUT with start over all Function Processes when Rendering
