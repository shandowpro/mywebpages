Vue.js [11 Learn VueJs In Arabic - Directives - v-bind - v-on] Main Topics :

> Main Subjects in this lesson are as following :

  1. [v-bind:AttributeName] || [:AttributeName] -> To Attach an Attribute value from the Vue Model
  2. [v-on:EventName] || [@EventName] -> To Attach an event Listner Function or property from the Vue Model

[ 11 Learn VueJs In Arabic - Directives - v-bind - v-on] Concept:

=> Using Vue Directives {v-bind} & {v-on} to attach Attributes and Events Inside [HTML elements] directly in DOM , from v-model :
  • [v-bind:AttributeName = "Vue Property"] || [:AttributeName = "Vue Property"] =>
    • To bind an Attribute value from the Vue Model's Property
    • [v-bind] ShortHand => to write the {:} and the Attribute Name directly
    • [v-bind] => We can not use the { { } } of Vue property Model

  • [v-on:EventListnerName ="Vue Property || Vue Method()"] || [@EventListnerName="Vue Property || Vue Method()"] =>
    • To set a Vue Model Function/Method inside an HTML Event Listner
    • [v-on] ShortHand => to write the {@} and the Attribute Name directly
    • [v-on] => We can not use the { { } } of Vue Property/Function Model

Vue.js [11 Learn VueJs In Arabic - Directives - v-bind - v-on] Importance & Benefits :

[Directives] => Provides an Easy Methods to apply certain Insturctions on HTML Elements as an Element's Attribute Auto Functionality Responding , and therefore more Dynamicallay

[v-bind] => Critical in any Vue Application, when it is required to bind a Vue property value inside an HTML's Attribute

[v-on] => Critical in any Vue Application, to set an Vue Model Function/property inside a Html element Event Listner
