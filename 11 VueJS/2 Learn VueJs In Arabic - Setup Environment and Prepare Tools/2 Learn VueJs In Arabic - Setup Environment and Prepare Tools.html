=> Vue.js [Main Concept] :
  • It is a progressive javascript Framework working mainly with HTMl Css and JS
  • Created by [Even Yoo]

  • One of the most popular Frameworks [Angular.js {Handeled Google} && React.js {Handeled Facebook} && Vue.js]
  • It is very easy to switching between different types fo FrameWorks , due to is has the same concepts

=> Vue.js [Features] :
  1. [Light weight] file in comparison with other Frameworks
  2. [High Proffessiont] written script
  3. [Virtual DOM] => Using a Virtual Dom liabrary instead of the original DOM , to Handel all it's Processes and then compile the Virtual to the main DOM
  4. [Resuable Components] => They are a group of Elements to be used it easily in different positions and different Projects , with easy one control position
  5. [Directives] =>
    1. They are Built-in actions or processes [Hooks] provided by [Vus.js] , to be processed in projects
    2. Vue.js provides the ability to create your own Directives
    3. Directives => Can provides the dealing with the backside much more easy by the Client side Interface
    4. Example Directives {If-Condition} => Can provides the same idea of the condtion if when dealing with the [API] (like processing certain Action in case of returing certain value from the database -> Than saving codes in API )
  6. [Events Handling] => Providing the professiont and easy Mehtod to deal with the Js Events without using long scripts in DOM & JS
  7. [Animation Handling] => Providing the professiont and easy Mehtod to deal with the CSS Animations, without using the CSS Formating
  8. [Watchers] => Providing the Monitoring Method for all actions and Processes in the Projects && Auto Updating TOOL without using any codes
  9. [CLI] => Providing the Commanline Tool Interface and provides the power of the Commandline with easy tool , which is very usful in compling and Building projects
  10. [Routing] =>
    1. It is A Proffessiont and easy Tool or Method to Navigation inside the project
    2. An Excellent Tool to deal with Framworks like [Laravel] & [YEE Framework] && [PHP Cake]
  11. [Unit testing CLI] => A great tool provided by Vue to implement the Unit Testing
  12. [Mixins] => (Exited in other Libraries like SASS && OOP)
    1. A practical Method to deal with group of elemnt properties, as a reusable group to be implement on other element, with an ability to modify and Addition
