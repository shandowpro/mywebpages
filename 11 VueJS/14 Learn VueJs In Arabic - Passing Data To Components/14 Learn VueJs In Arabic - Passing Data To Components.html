[ 14 Learn VueJs In Arabic - Passing Data To Component] Concept:

=> [Passing Data To Components] Concept :
  • This is providing the best pracise for passsing through datasources of the Main Vue Model => to the Vue Component data source
  • Passing Data through the Main Vue Model Data source => to the Vue Component Datasources Requires following :
    • Defining the Instances in the main Vue model Data Source => data: {}
    • Adding the Defined instances as an Array Elements => with in Vue Component Data Declearation => props : ['prop1' , prop2']
    • Using the Passed props or instances with in The Vue [Compoent Template]
    • Binding the Instances in to the new Component HTML Element with in the Vue contianer , whether the property is a {single} or Element of an {array} or an Element of {Object} or an Array of Objects =>
      • Bining a Single property => < componentName :prop1="prop1" >
      • Bining a Single property of an Array => < componentName :prop1="arraName[#index]" >
      • Bining a Single property of an Pbjeyt => < componentName :prop1="objName.propName" >
      • Bining a Single property of an Pbjeyt => < componentName :prop1="objName.propName" >
