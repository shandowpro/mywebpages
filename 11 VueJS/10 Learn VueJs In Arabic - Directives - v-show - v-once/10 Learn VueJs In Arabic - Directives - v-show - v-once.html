Vue.js [10 Learn VueJs In Arabic - Directives - v-show - v-once] Main Topics :

> Main Subjects in this lesson are as following :

  1. [v-once] -> To make the element static [make vue render for this element only for one time]
  2. [v-show="condition"] -> To make an element be showen [display: show] if the condition is True || Hide the Element if the condition is False

[ 10 Learn VueJs In Arabic - Directives - v-show - v-once] Concept:

=> Using Vue Directives within Vue Instances [properties && Methods] Concept :
  • [v-once] =>
    • To make the element static [make vue render from DOM for this element only for one time]
    • Suits the Static Element in many application

  • [v-show] =>
    • To make the Element be displayed only if the condition has return True
    • Suits the Toggling Element such as [Element to be toggled between Show(display:show) and Hide(display:none) ]
    • Differences between the {v-show} && {v-if} :
      1. In Case of [True] Condition => Both kinds of Directives Will Display the value of the Vue Property
      2. In Case of [False] Condition =>
        • {v-show} => This Directive will Only Toggle the (Display) Property to (None) Value
        • {v-if} => This Directive will Totally Remove the Element from the HTML script

Vue.js [10 Learn VueJs In Arabic - Directives - v-show - v-once] Importance & Benefits :

[Directives] => Provides an Easy Methods to apply certain Insturctions on HTML Elements as an Element's Attribute Auto Functionality Responding , and therefore more Dynamicallay

[v-once] => Critical in any Vue Application, when it is desired to Render the element value from the Vue for only one time

[v-show] => Critical in any Vue Application, to Toggle between the element Property [display: show - none]
