[Global Vs Local Components] Concept:

=> [Global Vs Local Components] Concept :
  • The {Local component} => Is a located registration / Scope , which will NOT be displayed in normal Vue Component, unless it will be Defiened and Called
  • The main Purpose of creating the Local component => For the limited/uncommon Usage Copmponent , which will be called for certain position
  • Main Differences between {Global Component} && {Local Component} following :
    • {Global Component} => Can be Called and Accessed with General Scope from any location in the Vue Application
    • {Local Component} => Can be Called and Accessed with Certain Scope from specified location in the Vue Application [If only it has been Added in to the Main Vue Model]
