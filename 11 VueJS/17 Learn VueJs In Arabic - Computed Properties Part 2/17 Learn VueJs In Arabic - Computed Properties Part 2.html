[Computed Properties] Concept:

=> [Computed Properties] Concept :
  • This Vue Instance will help Increasing Performance Rate by Saving Repeating all Method procedures by cashing he return value in serperated data space to directly display the value with out running all method's procedures again every Randering Time
  • Defined Methods in {Comptued Instance} can be Called in the Vue HTML Elements as [Variables]
  • Most important Difference in Performance between {Computed Methods} & {Normal Methods}
    1. Called in Vue HTML Element Syntax :
      • {Normal Methods} => In case of change any Property value in Vue Model (even it is NOT one of the Method's Independencies) => The whole Method will start over each Time The page Randered =>
        Bad Performance (specially with large Scale applications and Multiple Processes).
      • {Computed Methods} => In case of change any Property value in Vue Model => The Method's Independencies will be External Stored and be Displayed Every time the user Change Data=>
        Savning Time and Memory => Best Performance.
  • In this lesson => More Advanced Exmples of Using [computed{}] + some topics of using [ES6] in Syntax
