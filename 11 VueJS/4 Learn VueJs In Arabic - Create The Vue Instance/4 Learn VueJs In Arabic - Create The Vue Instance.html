[ 4 Learn VueJs In Arabic - Create The Vue Instances] Concept:

=> [Vue Instances] Concept :
  • It is Main Vue Sytnax elements
  • Vue main Syntax as following [in order] :
    1. Main Vue Container :

      It is the HTML To include all Vue Application => based on the [SPA] Concept

      This Container will be used and Been Accessed by The Vue Application by using the Container Element [Id]

    2. Vue Importing Framework :

      It will using the HTML [< script >] and the [src] inside it by the location of the Vue file

      The Vue file included in the Src attribute is a JS file

      The Syntax will be as =>
      <script src="Vue File Name"> </script>

    3. Vue Main Model:

      This is the Main Data of the Vue Framework

      This Model must include the folowing :

      1. The Vue Application Container Name (as Object Property Value of the {el:} Property Name)
      2. The Vue Application Data (as Object value of the {data:} Property Name)

  • Method of Accessing and Writing the Vue code inside the HTML

    Just using double Curly Brackets { { Vue code to be written } } [Mostash]

    Inside the double Curly Brackets { { Any Javascript || Vue Model Data Property Name can written and processed } }
