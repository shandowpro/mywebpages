[ 12 Learn VueJs In Arabic - Directives - v-model] Concept:

=> [v-model] Concept :
  • [v-model= "Vue Property"] =>
    • To Bind (Two Bind Method) to a HTML Element's Attribute value from the Vue Model
    • Similer to the {v-bind} , but with using (Two Ways Method Binding)
    • It is one of the most important and usual Directives used in Vue Application
    • {v-model} => Deals with (3) main HTML elements :
      1. {< input >} => Will deal with (Value) , whether the input type is [Text (Value), Checkbox (Checked Value), Radio Buton (Checked Value) ]
      2. {< select >} => Will deal with (Selected) value of Select Datasource
      3. {< textarea >} => Will deal with (Value) of Textarea
    • {v-model} => Deals with (3) Modifires :
      1. {.lazy}
      2. {.number}
      3. {.trim}

  • Differences between [One Way Binding Method {v-bind}] && [Two Way Binding Method {v-model}] =>
    1. {v-bind} [One Way Binding Method]

      Binding the Vue Property only FROM the Vue Model TO the HTML Element, and Vue Model Will Not respond to the Changing in Binded Value

    2. {v-model} [Two Way Binding Method]

      Binding the Vue Property in Both FROM the Vue Model TO the HTML Element and Vise Versa, with integrated repondness to the Changing in Binded Value

Vue.js [12 Learn VueJs In Arabic - Directives - v-model] Importance & Benefits :

[Directives] => Provides an Easy Methods to apply certain Insturctions on HTML Elements as an Element's Attribute Auto Functionality Responding , and therefore more Dynamicallay

[v-model] => Critical in any Vue Application, when it is required to Two Ways bindind Method a Vue property value inside an HTML's Attribute and Vise Versa

[v-model] => provides auto responding to the changes in the binded Vue Instances, so it is affecting Vue model itself
