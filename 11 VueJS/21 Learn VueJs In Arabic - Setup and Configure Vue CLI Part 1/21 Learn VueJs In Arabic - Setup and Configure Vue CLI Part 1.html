[Setup and Configure Vue] Concept:

=> [Setup and Configure Vue] Concept :
  • Downloading and installing {cmder} => to use the vue-cli on it, which is having following Features :
    1. Has modern features which are in the recent code Editor , and much more handy and pracitcal than the command line interfaces in the linux & mac & windows
    2. Include the Theme features to change the font & background properties and color
    3. Include the ability to create and operte several pages in the same time
    4. Include the more easy and handy order controlings
  • Setup the Vue CLI [Command line Interface] , through the following steps :
    • Checking for the Node version through the command line : [ npm --v ]
    • Checking if there is a previous version of the Vue CLI setup in this machine : [ npm vue-cli --v ]
    • Installing the vue-cli if it is not existed in the machine : [ npm install vue-cli ]
    • Accessing the required Directory => To Create the porject in : [ create vue project-Name{kabab-case} ]
      • { cd directoryName }=> To open the directory
      • { cd.. directoryName }=> To go up level the directory
    • Determining the Project Pre-set Methods Configurations by selecting form the list :
      1. Default Presetting Method {babel, eslint}
      2. Manually Presetting Method by choosing from following items :
        1. [Babel] => Which is a Javascript Transpiler (Translator + Compiler) => To modern JS Sytnax code readable for the older browser
          => {Recommended to Select}
        2. [TypeScript] => Which is a superior Javascript
          => { X Not Recommended to Select}
        3. [Progressive Web Application]
          => { X Not Recommended to Select}
        4. [Router] => Deal with Vue page links and redirectives for inner page navigation the Current page
          => { Recommended to Select}
        5. [Vuex] => A State Management [Will be studied later ]
          => { Recommended to Select}
        6. [CSS pre-processors] => Supporting the CSs libraires like (Sass)
          => { Recommended to Select}
        7. [Linter & Formatter] => Activating the Syntax Correcting & providing the [ Unified Standarized Formatter ]
          => { Recommended to Select}
        8. [Unit Testing] => Checking for Responsive of the Script of code & reduce the bugs & and Recovering for the page issues (Will Be studied later )
          => { NOT Recommended to Select for Now}
        9. [E2E] => Also one of the Testing pacakage
