ReactJS [5 Creating second inner section of Header] Main Topics :

> Main Subjects in this lesson will be => starting creating the second inner section of the Header inside the main page using the {MUI} , as following :

Main Subjects :

  1. Main points of this lesson , will include the following steps and points :
    1. Creating the second part of [Header] main component {Header2.jsx}, which inlcuidng the following inner (3) parts :
      • Shopping Cart Icon inner section
      • Search Box & inner menu list [Categories] inner section
      • Personal Icon [user settings] + Shopping cart Item inner section
      • Vip Notes


5 Creating second inner section of Header concept :

A] Adding {Shopping Cart icon + text Typography} icon [1] inner sectoin , as following steps :

{ShoppingCartOutlined + Typography} :

  1. {ShoppingCartOutlined} It is one of MUI icons type insdie '@mui/icons-material' , and it is being used to add the shopping Cart icon
  2. {Typography} Texual Icon , and it can be importerd from "@mui/material/Typography"
  3. The main steps and requirements to use {ShoppingCartOutlined} & {Typography} as following => inside the {Header2.jsx} :
    • Importings :
      1. {Box} => MUI element form '@mui/material' library =>

        The main parent contianer in [MUI] to hold all other child container and elemtns isnide it

        will be assgined with some properties with 'sx' attribute


      2. {Stack} => MUI element form '@mui/material' library => to as an inner child container of the whole scripts , with the folloWing aatributes :
        • direction={"row"} => beosue the default [display : flex] proeprty with value of direction = 'column'
        • alignItems={"center"} => to align all inner items inside this [Stack] with cetner of the row

      3. Typography => MUI element form "@mui/material/Typography" library => [it's script can be found inside the formal site of MUI comonents]
      4. < Box > => will be used as a Horizontal seperator container [inbetween space] between two {Typography} icons used in this script , by assignng this property of : flexGrow={1}

    • Inner styling attibute sx = { { propertyName : 'propertyValue' } } for all of elments to assign some css propreties




B] Adding {Menu List - SelectedMenuItem} icon [1] , as following steps :

{List | ListItem | ListItemText | MenuItem | Menu } elements :

  1. It is one of MUI Menu icons , and has inner list of menu items
  2. MUI Menu icons has several types menues , but we will use {SelectedMenuItem} in this script
  3. The {Menu List - SelectedMenuItem} icon will be placed after {Typography} icons
  4. some of inner elements of {Menu List - SelectedMenuItem} icon will use some of react classes such as [defined states , defined and imported functions ]
  5. The main steps and requirements to use this type of icons as following => inside the {Header1.jsx} :
    • Importings :
      1. The First section [Menu body] => including the following elements :
      2. {List} => MUI element from '@mui/material/List' library =>

        The parent container of the Menu body element in [MUI] to hold all other child container and elemtns isnide it

        will be assgined with some properties with 'sx' attribute


      3. {ListItem} => MUI element from '@mui/material/ListItem' library => to as an inner child container of the whole scripts , with the folloWing aatributes :
        • direction={"row"} => beosue the default [display : flex] proeprty with value of direction = 'column'
        • alignItems={"center"} => to align all inner items inside this [Stack] with center of the row

      4. {ListItemText} => MUI element from '@mui/material/ListItemText' library => to as an inner child container of the whole scripts , with the following aatributes :
        • direction={"row"} => beosue the default [display : flex] proeprty with value of direction = 'column'
        • alignItems={"center"} => to align all inner items inside this [Stack] with cetner of the row

        This Element will include {ExpandMore} => which is the down arrow after the list item value , which is being imported from the '@mui/icons-material' sub library , and assigned by some css properties



      5. Second section [Menu data list] , and including the following elements :
      6. {Menu} => Represent the Menu data list , and it is one of MUI element from '@mui/material/Menu' library => to as an inner child container of the it's child element , and has the following attributes :
        • id="lock-menu" => to act as the id of data list if the menu above
        • open={open} => to assing the status of the menu as open
        • anchorEl={anchorEl} => to call the defined status above =[ which is the data store of the item selected from the menu form the data list ]
        • onClose={handleClose} => to call the defined function => [ which is handle the closing event ]
        • MenuListProps => which is an object of properties related to te menu , as following
          1. "aria-labelledby": "lock-button",
          2. role: "listbox",

        This element will include the folowing children elements :

        1. Direct JS of mapping the defined array [options] of the menu data list , to return the next element:

          {MenuItem} =>

          1. which is the result of selecting from the menu data list
          2. This mapped will include a callback funtion with (2) parameters : [option, index]
          3. This element will has the next properties :
            • key={option} => [assign the mapping result inside teh key property ]
            • disabled={index === 0} => [we will delete it to remove the disabling option from the menu]
            • selected={index === selectedIndex} => [to assign the selected item from the data list of the menu inside the index vairable ]
            • onClick={(event) => handleMenuItemClick(event, index)} => [calling the function the handling the process of the clicking on the menu items list ]
            • sx= { { fontSize: "11px", p: "3px 10px", minHeight: "10px" } } => [some css properties for the data list items ]


      7. {MenuItem} => ReMUI element from '@mui/material/MenuItem' library => to as an inner child container of the whole scripts , with the folloWing aatributes :
        • direction={"row"} => beosue the default [display : flex] proeprty with value of direction = 'column'
        • alignItems={"center"} => to align all inner items inside this [Stack] with cetner of the row


    • Inner styling attribute sx = { { propertyName : 'propertyValue' } } for all of elments to assign some css propreties




C] Adding {Social Media} icons [3] , as following steps :

{ FacebookIcon , TwitterIcon , InstagramIcon } :

  1. These are type of the UI [icons-material] sub library
  2. These icons will be placed at the last of the {Stack} child container inside the header1
  3. The main steps and requirements to use this type of icons as following => inside the {Header1.jsx} : :
    • Importings :
      1. import TwitterIcon from "@mui/icons-material/Twitter"
      2. import FacebookIcon from "@mui/icons-material/Facebook"
      3. import InstagramIcon from "@mui/icons-material/Instagram"

    • Inner styling attibute sx = { { fontSize: "16px", color: "#FFF" , margin:"3px" } } for each of social media icon elements to assign some css propreties




D] Full final script of {Henader1.jsx} after adding all above elements , as following steps :
// A] First section [ Importing ] :
// 1- General Importing :
import React from "react";
import { useState, useContext } from "react";

// 2- Importing [dark/light] mode + other elements from the [mui/material] :
import { ColorModeContext } from "../../theme";
import { Box, Container, IconButton, Stack, useTheme } from "@mui/material";
import { DarkModeOutlined, ExpandMore, LightModeOutlined } from "@mui/icons-material";

// 3- Importing the [socialmedia] icons from the [mui/icons-material] library :
import TwitterIcon from "@mui/icons-material/Twitter";
import FacebookIcon from "@mui/icons-material/Facebook";
import InstagramIcon from "@mui/icons-material/Instagram";

// 4- Importing the [SelectedMenuItem] icons library :
import List from "@mui/material/List";
import ListItem from "@mui/material/ListItem";
import ListItemText from "@mui/material/ListItemText";
import MenuItem from "@mui/material/MenuItem";
import Menu from "@mui/material/Menu";
import Typography from "@mui/material/Typography";

// B] After the importing and before functional component {Header1()} :
// Required array variable for the [MemuList data] :
const options = ["Ar", "En"];
// C] Inside the functional component {Header1()} & before the return :
// Required variables for the dark/linght mode :
const colorMode = useContext(ColorModeContext);
const theme = useTheme();

// Requried defined states variables and functions for the [Menu List item ] :
const [anchorEl, setAnchorEl] = useState(null);
const [selectedIndex, setSelectedIndex] = useState(1);
const open = Boolean(anchorEl);
const handleClickListItem = (event) => {
setAnchorEl(event.currentTarget);
};
const handleMenuItemClick = (event, index) => {
setSelectedIndex(index);
setAnchorEl(null);
};

const handleClose = () => {
setAnchorEl(null);
};
// Inside the functional component and inside the [return] :
// This is the main contianer of the whole component's return :
< div >
< Box
sx = { {
bgcolor: "#2B3345", py: "4px", BorderRadiusBottomleft: 4, BorderRadiusBottomright: 4, } }
>

< Container >
< Stack direction={"row"} alignItems={"center"} >
< Typography >
< /Typography >
< Typography >
< /Typography >
< Box flexGrow={1} > < /Box > // This section is the [dark/light] mode :
< div > < /div > // This section is the [Menu List item] {2 main elements } :
// 1- the first part {List}
< List
sx= { { p: 0, m: 0 } }
component="nav"
aria-label="Device settings"
>
// This is the inner child element :
< ListItem
id="lock-button"
aria-haspopup="listbox"
aria-controls="lock-menu"
aria-label="when device is locked"
aria-expanded={open ? "true" : undefined}
onClick={handleClickListItem}
sx = { { "&:hover": { cursor: "pointer" } , color:"#fff" } }
>
< ListItemText
secondary={options[selectedIndex]}
sx = { { ".MuiTypography-root": { fontSize: "10px", color: "#fff" }, } }
/>
// This is the icon of the down arrow :
< ExpandMore sx ={ { fontSize: "10px", color: "#fff" } } />
< /ListItemText
< /ListItem >
> < /List >

// 2- the second part {Menu} :
< Menu
id="lock-menu"
open={open}
anchorEl={anchorEl}
onClose={handleClose}
MenuListProps= { {
"aria-labelledby": "lock-button",
role: "listbox"
} }
>
{options.map((option, index) => (
< MenuItem
key={option} selected={index === selectedIndex}
onClick={(event) => handleMenuItemClick(event, index)}
sx= { { fontSize: "11px", p: "3px 10px", minHeight: "10px" } }
>
{option}
< /MenuItem >
))}
< /Menu >

// Social Media Icons :
< FacebookIcon sx = { { fontSize: "16px", color: "#FFF" , margin:"3px" } } />
< TwitterIcon sx = { { fontSize: "16px", color: "#FFF" , margin:"3px" } } />
< InstagramIcon sx = { { fontSize: "16px", color: "#FFF" , margin:"3px" } } />
< Stack >

< /Container >
